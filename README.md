# blockchain-immutable-log
Децентрализованная система неизменяемого аудита критических событий ИТ-инфраструктуры (ImmutableLog).

*   **Мухаммет** создает и меняет файлы **ТОЛЬКО** в папке `ui_app/`.
*   **Лиза** создает и меняет файлы **ТОЛЬКО** в папке `blockchain/`.
*   **Артём** пишет тексты и кладет вордовские документы **ТОЛЬКО** в папку `docs/`.
