# 🛡️ ImmutableLog - UI & Audit Module

Этот модуль представляет собой интерфейс системы аудита логов на базе блокчейна. 
Разработан в рамках студенческого проекта (Факультет ИБ).

## 🚀 Стек технологий
- **Python 3.12+**
- **Streamlit**: Фреймворк для UI/Dashboard
- **Poetry**: Управление зависимостями и окружением
- **Pytest + Streamlit AppTest**: Автоматизированное тестирование
- **Docker & Docker-compose**: Контейнеризация

## 🛠️ Установка и запуск

### С использованием Poetry (локально)
1. Убедитесь, что у вас установлен Poetry: `pip install poetry`
2. Установите зависимости:
   ```bash
   poetry install
   ```
3. Активируйте окружение и запустите приложение:
   ```bash
   .venv\Scripts\activate  # Для Windows
   streamlit run src/app.py
   ```

### С использованием Docker
```bash
docker-compose up --build
```
Приложение будет доступно по адресу: `http://localhost:8501`

## 🧪 Тестирование и Качество (AQA)
Проект покрыт тестами на двух уровнях: Unit (ядро хэширования) и Integration (UI-сценарии).

- **Запуск тестов:** `pytest`
- **Проверка покрытия:** `pytest --cov=src`
- **Статический анализ (Mypy):** `mypy src tests`
- **Линтинг (Ruff):** `ruff check .`

## 📂 Структура папки
- `src/core/`: Бизнес-логика (расчет SHA-256, мок-сервис блокчейна).
- `src/app.py`: Точка входа в UI Streamlit.
- `tests/fixtures/`: Эталонные и скомпрометированные файлы логов для тестов.
